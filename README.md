# 🌿 FiveM Weed Lab HUD

A clean plant monitoring HUD designed for FiveM weed lab roleplay scenarios.  
Displays real-time plant data — water level, fertilizer, growth stage and harvest timer —  
in a compact glassmorphism panel overlay.  
Built with pure **HTML / CSS**, no dependencies.

> ⚠️ Design only — JS and Lua integration coming soon for full FiveM compatibility.

## ✨ Features

- Plant identity (name + number)
- Quality grade badge (B+, A, S...)
- Live gauges — Water, Fertilizer, Growth
- Harvest countdown timer
- Keybind reminders (E / F)
- Semi-transparent glassmorphism panel
- Clean typography — Patrick Hand + Space Mono

## 🛠️ Stack

`HTML` `CSS` `FiveM NUI`
